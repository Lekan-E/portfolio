Analysis on a Sport's Team League Season

An analysis of over 130,000 event data on Bayer Leverkusen's invincible season using Statsbomb API in Python and Tableau. After going an entire 34-game season unbeaten, I decided to obtain insights into how Bayer Leverkusen were able to achieve this. This process involved studying their formation, passing network, how well they shot the ball against each team using a goal map and many more.
